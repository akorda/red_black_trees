# Red-Black Trees

The aim of this repository is to host a reference implementation in C# of the ideas presented in the [associated paper](https://github.com/akorda/red_black_trees/blob/main/docs/paper/red_black_trees.pdf) about Red-Black trees also hosted here. The *source code* of the paper, written in LaTeX is also available.
